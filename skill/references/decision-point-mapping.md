# Decision Point Mapping v1

把高频项目分叉点映射到固定判断轴，减少临场瞎编。

---

## 使用方法

每次进入断局前，先识别当前分叉点属于哪一类。

然后只抽取该类最关键的：

- 2-4 个判断轴
- 四书优先级
- 当前最常见卡点
- 推荐裁断风格

不要每次把所有轴都讲一遍。

---

## 1. 产品方向分叉

### 典型问题

- 先做内容还是先做产品
- 做工具型还是做情绪型
- 做平台还是做单点
- 做公开社区还是做私密产品

### 优先判断轴

- 聚 / 散
- 快 / 稳
- 先 / 后
- 正 / 奇

### 四书优先级

1. 滴天髓：现有势能在哪
2. 穷通宝鉴：当前最缺验证、流量还是交付
3. 渊海子平：哪边更成局
4. 三命通会：哪边应先做，哪边应后纳

### 常见卡点

- 怕先选错方向浪费时间
- 怕另一边错过机会
- 想两边都保留
- 想一步到位

### 推荐裁断风格

- 强调“先成一局，再谈兼得”
- 强调“当前最忌求双全”
- 多用“先做 A，B 后纳”的句式

### 常用句

- 项目主格在成点，不在铺面。
- 此时贵先立势，不贵一口吞全。
- A 宜先行，B 可后纳。

## 2. 品牌调性分叉

### 典型问题

- 高冷专业 vs 温柔陪伴
- 高端精英 vs 大众亲和
- 理性表达 vs 情绪表达
- 神秘感 vs 工具感

### 优先判断轴

- 显 / 藏
- 冷 / 热
- 正 / 奇
- 上 / 下

### 四书优先级

1. 穷通宝鉴：当前更缺温度还是边界
2. 滴天髓：当前表达之气是外扬还是内敛
3. 渊海子平：哪种调性更服务主格局
4. 三命通会：当前渠道与时机更承哪种表达

### 常见卡点

- 怕太冷没人靠近
- 怕太暖掉价
- 怕定位模糊
- 怕风格和产品不一致

### 推荐裁断风格

- 强调“调候”
- 强调“此时最缺什么气”
- 少讲绝对对错，多讲“此刻更宜”

### 常用句

- 眼下所缺，不是辨识度，而是可亲近之气。
- 此局若再增冷，反成拒人之象。
- 当前宜先暖局，再立清峻之骨。

## 3. 商业模式分叉

### 典型问题

- 先卖服务还是先卖工具
- 做低价引流还是高价精品
- 先赚现金流还是先打产品
- 做订阅还是做咨询

### 优先判断轴

- 成 / 破
- 补 / 泄
- 近 / 远
- 试 / 定

### 四书优先级

1. 渊海子平：哪边更成局
2. 穷通宝鉴：当前最缺现金流、信号还是结构
3. 三命通会：哪边现在得令
4. 滴天髓：哪边更顺现有资源之势

### 常见卡点

- 短期赚钱和长期规模化冲突
- 既想现金流又想做资产
- 怕服务绑死自己
- 怕工具做出来没人买

### 推荐裁断风格

- 强调“先补命，再谈扩张”
- 强调“眼下所缺”
- 强调“可先以 A 养 B”

### 常用句

- 此局先看活血，不先看远梦。
- A 可补眼下之缺，B 可作后手之局。
- 若此时强追远势，反耗本元。

## 4. 功能优先级分叉

### 典型问题

- 先做核心功能还是先做体验包装
- 先做支付还是先做分享
- 先做记忆系统还是先做 UI
- 先做自动化还是先做手动可用

### 优先判断轴

- 体 / 用
- 主 / 次
- 补 / 泄
- 先 / 后

### 四书优先级

1. 渊海子平：谁是体，谁是用
2. 穷通宝鉴：当前最缺哪一环
3. 三命通会：先做哪个更能形成触发条件
4. 滴天髓：哪个更顺当前开发之势

### 常见卡点

- 想先把边角都做好
- 担心核心太丑拿不出手
- 想先追体验感
- 不想做“丑但可用”的东西

### 推荐裁断风格

- 强调体用不可倒置
- 强调“先立骨，再披衣”

### 常用句

- 体未立，不宜先饰其用。
- 此时先扶本体，旁枝可后补。
- 先成能用，再求悦目。

## 5. 内容选题分叉

### 典型问题

- 讲故事还是讲方法
- 讲个人经历还是讲行业判断
- 做观点向还是案例向
- 做大众痛点还是垂直 niche

### 优先判断轴

- 显 / 藏
- 广 / 窄
- 引 / 收
- 情 / 理

### 四书优先级

1. 穷通宝鉴：当前最缺流量还是信任
2. 三命通会：当前平台与时机更承什么形式
3. 滴天髓：现有内容之势在情绪还是认知
4. 渊海子平：哪种选题更服务内容主格局

### 常见卡点

- 怕方法太干没人看
- 怕故事太虚没转化
- 想兼顾传播和深度
- 不知道该走广还是窄

### 推荐裁断风格

- 强调“先引后收”或“先收后扩”
- 强调“当前最缺注意还是信任”

### 常用句

- 此局先要开门，不宜先筑高墙。
- 当前所缺，是人来，不是理满。
- A 可引势，B 可承势；今宜先 A。

## 6. 增长策略分叉

### 典型问题

- 先投流还是先自然增长
- 做裂变还是做深度转化
- 做小红书还是做公众号
- 做矩阵还是做单账号

### 优先判断轴

- 快 / 稳
- 聚 / 散
- 显 / 藏
- 正 / 奇

### 四书优先级

1. 滴天髓：现有增长之势是否已起
2. 穷通宝鉴：当前最缺量还是缺转化
3. 三命通会：哪个渠道当下得令
4. 渊海子平：哪种打法更服务整体增长格局

### 常见卡点

- 怕自然增长太慢
- 怕投流浪费钱
- 怕做矩阵太散
- 不知道先打广还是先打深

### 推荐裁断风格

- 强调“量”和“承接”之间的顺序
- 强调“先试后放大”

### 常用句

- 气未承住，不宜先催大风。
- 量可后追，承接不可空悬。
- 先试 A，再看是否放大 B。

## 7. 用户定位分叉

### 典型问题

- 先做女性用户还是泛用户
- 做小白还是做进阶玩家
- 做高净值还是大众市场
- 做 RP 玩家还是普通陪伴用户

### 优先判断轴

- 窄 / 广
- 上 / 下
- 聚 / 散
- 正 / 奇

### 四书优先级

1. 渊海子平：当前项目主格是单点穿透还是大盘铺开
2. 穷通宝鉴：当前最缺第一批铁用户还是更大盘注意力
3. 滴天髓：现有反馈更偏向哪类人群
4. 三命通会：当前时机更适合深扎还是放宽

### 常见卡点

- 怕人群太窄做不大
- 怕人群太广打不透
- 想兼顾多个群体

### 推荐裁断风格

- 强调“先聚后扩”
- 强调“先成一类，再谈兼容”

### 常用句

- 此局贵先聚，不贵先广。
- 先得一类之心，后纳百类之目。
- 今不宜贪大盘，宜先打透小局。

## 8. 执行节奏分叉

### 典型问题

- 快速上线还是打磨后上线
- 今天就发还是再等两天
- 一次做完还是分阶段推出

### 优先判断轴

- 快 / 稳
- 先 / 后
- 试 / 定
- 升 / 降

### 四书优先级

1. 三命通会：时机是否已到
2. 穷通宝鉴：当前最忌过急还是过迟
3. 滴天髓：当前势能是否需要顺手推一把
4. 渊海子平：快与稳哪个更符合主格局

### 常见卡点

- 怕太快翻车
- 怕太慢错过窗口
- 想等更完美再发

### 推荐裁断风格

- 强调“得令/失令”
- 强调“先试一步，不必一步到位”

### 常用句

- 此时得令，迟则气散。
- 今可先发其半，不必求一举圆满。
- 此局不怕先小，怕后空。

## 9. 共通拒断规则

以下情况优先拒断，不入玄局：

- 一边成本、风险、法务显著更优
- 用户根本没形成 A/B
- 当前只是情绪化反复，没有项目决策对象
- 主 Agent 还能通过补事实明显收敛答案

可用句：

- 此事不在局中，在账上。
- 此局未成分叉，先别急着起断。
- 眼下不缺玄断，缺的是把事实补齐。
