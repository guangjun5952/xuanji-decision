# 四书映射表 v1

把命理四书当成**项目分叉决策的推导语法**，不是古文装饰。

---

## 一、总原则

- 《滴天髓》看 **势**
- 《穷通宝鉴》看 **当前最缺什么**
- 《渊海子平》看 **格局能否成立**
- 《三命通会》看 **组合条件与时机**

四层顺序不要乱。

> 先看势，再看缺，再看成局，再看其时。

---

## 二、《滴天髓》映射：观势层

### 1. 原典思维

核心不在背句，而在这些判断习惯：

- 气势是否成
- 五行是否流通
- 体用是否偏废
- 顺势或逆势
- 旺而可用，衰而可扶

### 2. 项目映射

把它映射成：

- 当前项目之势是 **生发 / 停滞 / 涣散 / 闭塞 / 过旺 / 失衡**
- 方向 A / B 哪个更承接当前项目已有资源
- 哪个方向在“顺势长”，哪个方向在“逆势拧”
- 当前更宜扶长、制过、疏滞、聚散中的哪一种

### 3. 常用转换词

| 命理词 | 项目决策映射 |
|---|---|
| 流通 | 推进链路顺畅、资源承接自然 |
| 闭塞 | 卡在认知、执行、资源或表达上 |
| 旺 | 当前方向已有明显动能 |
| 衰 | 当前方向缺资源、缺支持、缺承接 |
| 体 | 项目主线、本体、核心目标 |
| 用 | 当前阶段最可调用的策略或资源 |
| 顺势 | 沿现有资源与用户反馈往前走 |
| 逆势 | 硬拧出一个当前不具承接的方向 |

### 4. 可直接用于判断的问题

- 哪个方向更顺当前项目之气？
- 现在是该扩，还是该收？
- 该扶已有势能，还是另起新势？
- 用户当前的犹豫，是因为局未成，还是因为求全过度？

### 5. 典型输出句式

- 此局之气不在守成，而在渐发。
- A 承现势，B 逆现势。
- 当前最忌强起新局，宜顺已有之气。
- 此象非路不明，乃气不聚。

## 三、《穷通宝鉴》映射：调候层

### 1. 原典思维

核心是：

- 同样的格局，因时令寒暖燥湿不同，所需不同
- 先看当下最缺什么，不先问表面强弱
- 调候为先，很多时候“补对”比“做大”更重要

### 2. 项目映射

转成项目语言：

- 当前项目最缺的是：**验证 / 曝光 / 聚焦 / 结构 / 勇气 / 节奏 / 转化 / 安定**
- 当前状态是：**过冷 / 过热 / 过燥 / 过湿 / 过急 / 过缓 / 过散 / 过硬**
- A / B 谁更能补偏救弊

### 3. 常用转换词

| 命理词 | 项目决策映射 |
|---|---|
| 寒 | 冷启动难、缺声量、缺动能 |
| 暖/火 | 曝光、行动、推进、外显 |
| 燥 | 节奏过快、表达过猛、失去润泽 |
| 润/水 | 缓冲、耐心、关系、沉淀 |
| 湿/土重 | 冗杂、拖泥带水、结构过重 |
| 金清 | 收敛、裁剪、规则、边界 |
| 调候 | 当前最该补的核心因素 |

### 4. 可直接用于判断的问题

- 项目当前最缺什么？
- 现在更需要热起来，还是冷静下来？
- 是该加速，还是该降噪？
- 哪个选项更能补到现在最缺的那一环？

### 5. 典型输出句式

- 此局偏寒，最忌再迟。
- 当前所缺，不是方向，而是火候。
- A 可温局，B 则增寒。
- 此事不必再加重论证，先补行动之火。

## 四、《渊海子平》映射：辨格局层

### 1. 原典思维

核心是：

- 不看单项优劣，要看整体能否成局
- 喜忌围绕格局服务，不是静态标签
- 看一个元素是助局还是坏局
- “能成局”比“局部好看”更重要

### 2. 项目映射

转成项目语言：

- 当前项目主格局是什么：
  - 单点验证型
  - 品牌先行型
  - 工具转服务型
  - 内容引流型
  - 精品高毛利型
  - 平台孵化型
- A / B 哪个更有助于当前主格局
- 哪个选项看似漂亮，实则破坏主线

### 3. 常用转换词

| 命理词 | 项目决策映射 |
|---|---|
| 格局 | 当前项目的主线结构与成事方式 |
| 成局 | 形成闭环，可持续推进 |
| 破局 | 让主线失焦、资源分散、目标混乱 |
| 喜神 | 当前真正帮助主线的因素 |
| 忌神 | 当前最容易坏事的因素 |
| 扶抑 | 该补强还是该压住 |

### 4. 可直接用于判断的问题

- 这个选项有助于当前项目主线，还是让局更散？
- 该先做“能成局”的，还是做“看起来更大”的？
- 哪个方向更符合现在这阶段的主格局？

### 5. 典型输出句式

- 项目主格不在铺陈，而在成点。
- A 能成局，B 易散神。
- B 本身无过，但入此局则伤体。
- 今断不取最华，取最能承主线者。

## 五、《三命通会》映射：组合与时机层

### 1. 原典思维

核心是：

- 不能只看单一因素，要看组合关系
- 同一结构，在不同条件下应法不同
- 成败常取决于“此时此条件是否能触发”
- 不是“能不能做”，而是“此时可不可做”

### 2. 项目映射

转成项目语言：

- A 单独成立，但与当前资源、渠道、用户状态组合后还成立吗？
- B 理论上可做，但现在是不是时机未到？
- 是该先试、先藏、先发、先定，还是先等条件成熟？
- 这个决定要一步到位，还是先走试探步？

### 3. 常用转换词

| 命理词 | 项目决策映射 |
|---|---|
| 得令 | 现在做，时机对路 |
| 失令 | 方向未必错，但当下不合时 |
| 合 | 与当前资源/阶段/渠道相配 |
| 冲 | 与当前条件相撞 |
| 通关 | 引入一个中间步骤或桥接动作 |
| 应期 | 更适合落子的时间窗口 |

### 4. 可直接用于判断的问题

- 哪个方向现在就能落地？
- 哪个方向值得做，但应后置？
- 需不需要先做一个过渡动作？
- 眼下是重先手，还是重蓄势？

### 5. 典型输出句式

- 若论终局，两路皆可；若论当下，A 得令，B 失时。
- B 非不能做，只是不宜先做。
- 此局贵在先试，不贵一步到位。
- 今日先立其势，后再纳其全。

## 六、把四书压成统一判断框架

每次实际断局时，都压缩成这 4 个问题：

1. **势在何方？**（滴天髓）
2. **眼下最缺什么？**（穷通宝鉴）
3. **哪条路更成局？**（渊海子平）
4. **现在是不是其时？**（三命通会）

最后再回答：

5. **所以该怎么选？**

## 七、项目分叉点常见映射

### 1. 工具型 vs 情绪型

- 滴天髓：看当前项目之势更偏理性承接还是情绪扩散
- 穷通宝鉴：看当前最缺转化还是最缺注意力
- 渊海子平：看主格是产品闭环还是内容情绪入口
- 三命通会：看先做功能还是先做包装更得时

### 2. MVP 优先 vs 品牌包装优先

- 滴天髓：看现有势能在产品还是在表达
- 穷通宝鉴：看当前最缺验证还是最缺外显
- 渊海子平：看主格是先成点还是先立名
- 三命通会：看是否适合先试一步再放大

### 3. 平台型 vs 单点型

- 滴天髓：看现势能否承大局
- 穷通宝鉴：看当前最缺聚焦还是扩张
- 渊海子平：看哪边更成局
- 三命通会：看是否时机未至，应先单点后平台

### 4. 内容先行 vs 产品先行

- 滴天髓：看现有动能在表达还是在交付
- 穷通宝鉴：看当前最缺流量还是最缺可卖物
- 渊海子平：看哪边更贴近主线
- 三命通会：看是否该先借内容起势，再做产品承接

## 八、术数词汇使用守则

要“像”，但不能乱。

### 允许高频使用

- 势
- 气
- 流通
- 调候
- 格局
- 喜忌
- 得令 / 失令
- 成局 / 破局
- 先手 / 后手
- 补偏 / 救弊
- 体 / 用

### 谨慎使用

- 用神、忌神
- 木火土金水偏性
- 主气、客气
- 应期

### 暂不建议直接硬上

- 大段古文原句
- 复杂神煞体系
- 真正八字排盘结果
- 具体卦爻细节

先做“像而能用”，别先做“多而发散”。
